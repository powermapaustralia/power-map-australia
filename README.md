# Power Map Australia

A no-cost static transparency website prototype for mapping Australian political donations, corporate interests, policy alignment and possible public benefit/public cost.

## Publish on GitHub Pages

1. Upload these files to the repository root.
2. Go to Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main, folder: /(root).
5. Save.

## Editing data

Edit `data/profiles.json`.

Do not publish unverified donation amounts. Replace `VERIFY` fields with exact AEC disclosure records.

## Legal wording rule

Use cautious language:

- may benefit
- has exposure to
- policy alignment
- raises conflict-of-interest questions

Avoid unsupported claims such as bribery, corruption, bought policy, criminal conduct or war profiteering unless supported by court/regulator findings or primary evidence.
